# FlaskWebsite
 
This is a basic website built with Python's Flask library that features an account system and an ability to store notes on the web, which can be edited pending usre interaction on the site. 

To launch this site, clone this repo, install via the requirements.txt file and run the main.py file. 
THe website will open in localhost, port 5000 for you to interact with the website. 

This project was primairly built to gain a deeper understanding of Python's Falsk library and web development in general. Some learning objectives inlcude:
- [x] Creating a basic website with Flask
- [x] Understanding Bootstrap classes 
- [x] Creating a database 
- [x] Managing user sessions and accounts 
- [x] Adding and deleting objects from the database 
- [x] Formatting webpages using JS, HTML and CSS
- [x] Understanding the Jinja framework
- [x] etc..
